- auto-mode (חד-פעמי) — CC רץ בלי אישור בין צעדים; classifier חוסם מסוכן.
  מתי: משימה auto-safe עם ביצוע ברור, בלי תנאי-לולאה.
- goal (לולאה) — CC ממשיך תור-אחר-תור עד תנאי-סיום; Haiku בודק כל תור.
  מתי: משימה auto-safe עם תנאי-סיום מדיד אובייקטיבי.
